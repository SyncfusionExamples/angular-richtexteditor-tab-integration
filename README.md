# Angular Rich Text Editor Tab Integration
An example demonstration of the EJ2 [Angular Rich Text Editor](https://www.syncfusion.com/angular-components/angular-rich-text-editor) with the Syncfusion Tab component integration

## Quick start

Prerequisites:

- Node.js 20.x (recommended) and Also supported: Node.js 22.x or 24.x
- npm

Install dependencies and run the app:

```bash
npm install
ng serve
```

Open http://localhost:4200/ in your browser.

## What this sample shows

- How to include the Rich Text Editor in an Angular component
- How to integrate the Tab component in the Rich Text Editor
